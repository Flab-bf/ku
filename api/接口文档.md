## 用户接口
- **注册接口（/register）：**
	-  请求方法：POST  
	-  请求参数：
		- nickname（string）：用户昵称，必填.  
		- account（string）：用户名，必填，需保证在唯一.  
		- password（string）：用户密码，必填.
- 请求示例：  
```
{
  "nickname": "test_nickname",
  "account" : "test_username",
  "password": "test_password"
}
```

- 响应示例  
	- 成功
```
		{
		"message": "注册成功"
		}
```
- 失败
```
	{
	"error": "用户名重复"
	}
```

- 响应状态码：
	-  200：注册成功.
	-  400：参数错误或用户名重复. 

- **登录接口（/login）：**
	-  请求方法：POST
	-  请求参数:
		-  account（string）：用户名，必填.
		-  password（string）：用户密码，必填.
	- 请求示例：
	```
	{
  "account": "test_username",
  "password": "test_password"
	}
	```
	- 响应示例
		- 成功
		- ```
		- {
  		"message": "登录成功"
			}
		 ```
		- 失败
		 ```
		{
  		"error": "用户名或密码错误"
		}
		```
	- 响应状态码:
		- 200：登录成功.
		- 401：用户名或密码错误.
